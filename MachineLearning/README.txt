Run with:
- python cooking.py <your_data.json file> 

Requirements: 
- R must be installed
- Python 3.5.1

The following needs to exist as named here:
- ingredients.txt
- test_data_sample.json
- script_files/

**This last one is a folder that files the script creates are placed in temporarily before they are deleted at the end of the program's successful completion.
